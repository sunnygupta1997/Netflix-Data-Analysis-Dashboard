Netflix Global Content Strategy Dashboard 📊
📌 Project Overview
This project involves a comprehensive analysis of a Netflix dataset containing 1,500 records of movies and TV shows. Using Advanced Excel, I transformed raw, unstructured data into a dynamic, multi-dimensional dashboard that provides key insights into content production, audience demographics, and regional trends.

🛠️ Data Analytics Workflow
Data Cleaning: Handled missing values and standardized category names for consistent reporting.

Data Modeling: Used Pivot Tables to aggregate large volumes of data across multiple dimensions (Genre, Country, Rating, and Year).

Visualization: Developed Pivot Charts to translate complex numbers into easy-to-understand visual stories.


🔍 Key Business Insights
Content Inventory: The library maintains a healthy balance between Movies (764) and TV Shows (736), supporting both one-time viewers and long-term subscribers.

Genre Saturation: Crime (178) and Drama (175) are the leading genres, followed closely by Comedy (162), indicating high market demand for these categories.

Audience Targeting: The majority of Netflix’s library is focused on TV-MA (262) and TV-14 (260) ratings, showcasing a clear focus on adult and teen demographics.

Release Velocity: The analysis tracks content growth from 2005 to 2024, showing significant production spikes in the last decade.

Regional Dominance: Detailed analysis for major markets including India, Canada, France, Germany, and South Korea.

🚀 Interactive Features
Dynamic Filtering: View specific insights by toggling countries or content types (Movie vs. TV Show).

Trend Tracking: Analyze how release volumes have changed over the last 20 years.

Genre Analysis: Breakdown of content variety to identify niche vs. mainstream categories.
